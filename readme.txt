Para este proyecto se ha desarrollado solo el lado del frontend,
utilizando HTML, CSS, BOOTSTRAP 4, el mismo es estructurado al estilo 
de facebook, por lo que el usuario a la hora de verlo estará familiarizado con 
el diseño. 

Nota: el proyecto ha sido desarrollado a manera de práctica, pero es totalmente 
operativo y adaptable a cualquier dispositivo, ya que el mismo mantiene un 
responsive desing.
